# [CSDM] VIP System

A **Counter-Strike 1.6** VIP system plugin that provides VIP players with an exclusive weapon selection menu, VIP status indication on the scoreboard, a custom VIP knife model, free VIP access for Steam players, and additional VIP features using **ReAPI**.

## Plugin Information

  - **Name**: [CSDM] VIP System
  - **Version**: 1.0
  - **Author**: DadoDz
  - **Game**: Counter-Strike 1.6

## Requirements
  - AMX Mod X 1.9+
  - ReAPI Module
  - ReGameDLL_CS

## Installation
1. Place ```cs_vip_system.sma``` in: ```addons/amxmodx/scripting/```
2. Compile it with your AMXX compiler.
3. Place the compiled ```.amxx``` file in: ```addons/amxmodx/plugins/```
4. Add this line to your ```plugins.ini```: ```cs_vip_system.amxx```
5. Restart your server.

## Commands

| Command | Description |
|-------|-------------|
| `/vm` | Opens the vip menu |
| `vm` | Opens the vip menu |
| `nightvision` | Opens the vip menu |
| `/vip` | Command displays the vip.html MOTD containing information about the VIP system |
| `vip` | Command displays the vip.html MOTD containing information about the VIP system |
